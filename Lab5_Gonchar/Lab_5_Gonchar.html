# Крок 1: Інтерактивне введення елементів масиву.
n = int(input("Введіть кількість елементів у списку: "))  
my_list = [int(input(f"Елемент {i + 1}: ")) for i in range(n)]

# Крок 2: Оптимізований алгоритм бульбашкового сортування.
def bubble_sort(arr):
    length = len(arr)
    for i in range(length - 1):
        swapped = False  # Оптимізація: перевіряємо, чи був обмін елементів.
        for j in range(length - 1 - i):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]  # Обмін елементів.
                swapped = True  # Якщо був обмін, встановлюємо прапорець.
        if not swapped:
            break  # Якщо на поточній ітерації не було обмінів, виходимо з циклу.

# Виклик сортування.
bubble_sort(my_list)

# Виводимо відсортований список.
print("Відсортований список:", my_list)

# Мій код
# Створення матриці 8x8 з порожніми клітинками, позначеними "_"
board = [["_" for _ in range(8)] for _ in range(8)]

# Розміщення турів у чотирьох кутах шахівниці
board[0][0] = "R"  # Верхній лівий кут
board[0][7] = "R"  # Верхній правий кут
board[7][0] = "R"  # Нижній лівий кут
board[7][7] = "R"  # Нижній правий кут

# Виведення шахівниці у зручному форматі
for row in board:
    print(" ".join(row))

У Python списки передаються за посиланням. Тому list_1, list_2 і list_3 вказують на той самий об’єкт у пам’яті.

Після першого del list_1 [0]: список стає [“B”, “C”]. Після другого del list_2 [0]: залишиться тільки [“C”].

# Завдання 2
list_1 = ["A", "B", "C"]
list_2 = list_1
list_3 = list_2

del list_1[0]  # Видаляємо "A", залишиться ["B", "C"]
del list_2  # Видаляємо лише посилання на список

print(list_3)
['B', 'C']

Після del list_2 посилання на список видаляється, але сам об’єкт списку в пам’яті залишається, оскільки ще існує list_3.

Після del list_1[0]: список зміниться на [“B”, “C”].

Оскільки всі змінні (list_1, list_2, list_3) вказують на той самий об’єкт, після очищення list_2 всі списки стануть порожніми. Вивід:[]
