Створіть словник, який містить інформацію про більше, ніж один номер телефону для кожного з контактів в телефонній книзі. Ключами будуть імена контактів, а значеннями - список їхніх номерів телефонів. Напишіть функцію, яка дозволяє додати новий номер телефону до списку номерів телефону певного контакту. Потім виведіть на екран список номерів телефонів для всіх контактів.

# Мій код 

# Створити словник з інформацією про контакти
phone_book = {
    "Олександр": ["123-456-7890", "098-765-4321"],
    "Марія": ["234-567-8901", "987-654-3210"],
    "Анна": ["345-678-9012", "876-543-2109"]
}

# Функція для додавання нового номеру телефону до контактів
def add_phone_number(name, phone_number):
    if name in phone_book:
        phone_book[name].append(phone_number)
    else:
        phone_book[name] = [phone_number]

# Додати нові номери телефонів
add_phone_number("Олександр", "112-233-4455")
add_phone_number("Іван", "667-788-9900")

# Вивести на екран список номерів телефонів для всіх контактів
for name, numbers in phone_book.items():
    print(f"{name}: {', '.join(numbers)}")
